# Syntecxhub_User_Management_System
The User Management System is a RESTful API project that performs CRUD operations for managing users. It uses MongoDB and Mongoose for database handling, implements authentication for security, and allows API testing through Postman to ensure smooth and reliable functionality.


# User Management System

A full-stack User Management System built using:

- Node.js
- Express.js
- MongoDB
- HTML/CSS/Bootstrap
- JWT Authentication

## Features
- User Login/Register
- Admin Dashboard
- User CRUD Operations
- Authentication & Authorization

## Installation

```bash
npm install
npm start
